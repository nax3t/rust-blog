# Rust Blog

A blog engine written in Rust.

## Development

### Prerequisites
- Rust (latest stable version)
- Cargo (comes with Rust)

### Building
```bash
cargo build
```

### Testing
```bash
cargo test
```

### Running
```bash
cargo run
```

## Versioning
This project follows [SemVer](https://semver.org/) for versioning. For the versions available, see the tags on this repository.
